# Personal Expense Tracker

A React Native Expense Tracker application built using Expo CLI.

## Features

* Add expenses
* Delete expenses
* Category selection
* Input validation
* Total expense calculation
* Color-coded categories

## Categories

* Food
* Transport
* Utilities
* Entertainment
* Other

## Technologies Used

* React Native
* Expo CLI
* Expo Router

## Run the Project

1. Install dependencies

npm install

2. Start the application

npx expo start

3. Open in Expo Go or browser

npm run web
