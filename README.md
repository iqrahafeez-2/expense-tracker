# Personal Expense Tracker

## Project Overview

This application allows users to add, view, and delete personal expenses while tracking the total amount spent. It is developed using React Native and Expo CLI as part of the Friendsware Solutions Summer Internship Program.

## Features

* Add expenses
* Delete expenses
* Category selection
* Input validation
* Total expense calculation
* Color-coded categories

## Categories

* Food
* Transport
* Utilities
* Entertainment
* Other

## Technologies Used

* React Native
* Expo CLI
* Expo Router

## Run the Project

### Install dependencies

npm install

### Start the application

npx expo start

### Open in browser

npm run web
